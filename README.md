# short_video-user-behavior-analysis
基于真实短视频用户行为数据，完成用户活跃时段、内容偏好、互动特征与用户分层的全流程数据分析，输出可落地的业务建议。
# 短视频用户行为分析 | 数据分析作品集

## 项目背景
本项目基于真实短视频APP用户行为数据，从用户活跃、内容偏好、互动特征及用户分层四个维度进行分析，为平台内容运营和创作者提供数据支持与业务建议。

## 数据说明
- 数据量：12万+条观看记录
- 字段：`user_id`、`video_id`、`video_category`、`like_type`、`relay_type`、`time`
- 数据状态：无空值、无重复、已完成清洗与异常值处理

## 核心分析结论
1.  **用户活跃时段分析**
    - 单日活跃最高峰出现在 **15点（下午3点）**，次高峰在早间8点，上午时段活跃度偏低。
    - 工作日观看量显著高于周末，且流量波动更大，高峰更明显；周末用户全天观看更平稳，无明显高峰。
    - 建议：创作者优先选择工作日的下午3点发布内容，其次是早高峰时段，以获取更高曝光。

2.  **内容偏好分析**
    - 播放量TOP3的视频分类为：美食（food）、新闻（news）、旅游（tourism），是平台的流量基本盘。
    - 音乐（musics）、健康（health）类内容的用户点赞率更高，用户粘性更强。
    - 建议：创作者可优先选择美食、新闻等大众赛道入局，或深耕音乐、健康等垂类赛道以获取高互动。

3.  **互动行为特征**
    - 用户互动以点赞为主，点赞率远高于转发率，用户主动传播意愿较低。
    - 建议：平台可通过转发抽奖、激励活动等方式，提升用户转发行为，促进内容裂变传播。

4.  **用户活跃度分层**
    - 样本用户中，重度用户（观看次数≥120）占比70.1%，中度用户（60~119次）占比29.9%，无轻度用户。
    - 用户整体活跃度极高，平台粘性强，核心用户占比高。
    - 建议：运营重点可放在通过专属内容和互动活动，进一步提升重度用户的参与感，同时引导中度用户向重度用户转化。
